Macbook Pro

Starting Price: $1999

I currently have the Macbook Pro 14in M1 Max with 32 GB of ram. This is the newest model by apple. This macbook is great, however I would be careful. The macbook itself is very powerful however it comes with a price, if you want the specs I have it would set you back to atleast $3000. After a month of using the Macbook, somethings stand out to me. I relizied how good the M1 Macbook Air is for the price, for $999 you can get a lot of power. The new macbook pro is tool for individuals that need that extra horsepower in GPU and CPU. After buying this, I feel like I would of been better off with the M1 Pro chip, this chip is balanced and cheaper. The M1 Max that I have is a bit too much powerful. I will most likely never use all 24 gpu core and 10 core cpu. However on daily task it is very snapy, the battery life is amazing aswell. If you do not use apps that are optimized for the m1 chips then you loose performance in those appilcation. If you are not a daily mac user using mac applications I would hold off and buy a good used intel macbook pro or wait until the apps you use get optimzied with these new powerful chips. The display,keyboard and speakers are amazing on this laptop. My personal review is that this is a great machine if you need the extra power but if you are just a casual user with maybe some intensive tasks the M1 air will do wonders.
